# minicurso-paperclip-skills-aula-2

Bônus exclusivo de quem fez a **Aula 2** do Mini-curso (Engenharia de Dados Academy): a skill que instala o Paperclip + um **starter pack de skills** pra você montar sua empresa digital com um "time" pronto no dia 1.

Cada skill é uma pasta com um `SKILL.md` (formato Claude Agent Skill). Funciona no Claude (chat, Cowork, Claude Code) e conecta como **Company Skill** no Paperclip.

## 🛠️ Instalação do Paperclip
- **instalar-paperclip/** — guia o Claude Code a instalar o Paperclip pra você (via `paperclip.ing/llms.txt`), detectando o sistema.

## 🎁 Starter pack de skills (bônus)

**⭐ Carro-chefe**
- **criadora-de-skills** — cria novas skills a partir de uma descrição (ensina a pescar).

**✍️ Copy / Conteúdo**
- gerador-de-post · roteiro-de-video-curto · titulos-e-headlines · reescritor-de-texto · resumo-de-conteudo

**🎨 Social / Design**
- estrutura-de-carrossel · calendario-de-conteudo · briefing-de-imagem

**💰 Vendas**
- script-de-abordagem · resposta-a-objecoes · follow-up-de-vendas

**📊 Financeiro básico**
- controle-de-gastos · precificacao-simples

**⚙️ Operações / SOP**
- criar-sop · onboarding-de-cliente · resumo-de-reuniao

**🙋 Marca pessoal**
- bio-e-perfil · planejador-de-autoridade

> As skills **triagem-de-email** e **rascunho-no-meu-tom** (Aula 1) estão no repo [minicurso-skills-aula-1](https://github.com/owshq-mec/minicurso-skills-aula-1).

## 📦 Como instalar uma skill
**No Claude:** Configurações → Capabilities → **Skills** → **Add** → selecione a pasta da skill (com o `SKILL.md`) → ligue o toggle.

**No Paperclip (Company Skills):** página **Skills** → cole a URL deste repositório → **Add** → anexe cada skill ao setor/agente que fizer sentido.

## ✅ Pré-requisito
No Claude, ligue **Configurações → Capabilities → "Execução de código e criação de arquivos"** (necessário para skills).
