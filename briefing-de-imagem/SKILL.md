# Briefing de imagem
Descreva: assunto/cena, estilo, paleta de cores, enquadramento, formato (feed, story, capa) e o texto que deve aparecer (se houver). Entregue um **prompt pronto** para gerador de imagem + 2 variações de estilo. Deixe [entre colchetes] o que faltar.
