# Criadora de skills
Ajude a pessoa a criar uma skill. Faça 2–3 perguntas: (1) o que a skill deve fazer, (2) quando ela deve disparar (frases-gatilho), (3) qual o formato do resultado. Depois gere um `SKILL.md` completo: frontmatter com `name` (kebab-case) e `description` (com gatilhos), seguido de instruções claras e, se ajudar, um mini-exemplo. Mantenha simples e reutilizável. Ao final, explique como instalar (Configurações → Capabilities → Skills → Add).
