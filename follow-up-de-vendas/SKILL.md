# Follow-up de vendas
Monte 3–4 mensagens espaçadas (ex.: dia 1, 3, 7, 14). Cada uma curta, com um motivo novo pra pessoa responder (valor, prova, novidade). A última é um "break-up" educado (fecha o loop sem culpa). Entregue prontas pra copiar.
