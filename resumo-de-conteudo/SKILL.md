# Resumo de conteúdo
Entregue: um resumo de 1 frase, 3–5 bullets com os pontos-chave e, se houver, os action items. Seja fiel ao conteúdo — não acrescente opinião nem invente dados. Se o texto for técnico, mantenha os termos corretos.
